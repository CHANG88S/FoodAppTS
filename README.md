# FoodAppTS

This is an in progress food app that I am currently build from scratch to learn the full stack development cycle

The idea came from Yelp's/Beli's downside for me. The fact that you could not rate individual items. Also not being able to rate half stars

A single item bad item should not determine a whole place by itself especially if you order something that could be their best or worst dish or a bad experience
